Qumulo Core - Out of Boxing Experience

Project Introduction

The UX team is redesigning the out of boxing experience(OOBE) for Qumulo Core product. The product is composed of two parts: the hardware hosting all the data and the software helping users manage the data. Previously, the out of boxing experience barely existed because the installation process is done by Qumulo customer support, which takes a lot of efforts and resources. The goal of this project is to optimize the installation process by enabling users to install and configure the product by themselves in an intuitive and easy way.

Company

Qumulo

Created

06/2014

Role

UX Designer

Why does out of boxing experience matter?

The user experience is more than the satisfaction users get from interacting with the software. It is about the comprehensive experience users get from purchasing, setting up the product, using it day to day, and seeking assistance from customer support.

For our product, the user's journey starts from the moment when they lift the packaging box. A good first impression is critical since the opinions users formed through first several interactions with the product will go a long way in building the trust between users and the brand.

Project Outcomes

Delivering out of box experience requires multiple departments collaborations since it is associated with product purchasing, ordering, packaging and delivering. In the end of this project, the UX team designed a tri-fold quick start guide to help users install the hardware and software by themselves, as well as made recommendation the formalized process recommendations across Sales, marketing, customer support, hardware and operations teams.

Process

1. User interview
2. Brainstorming
3. Stakeholder review meeting - concept R1
4. Sketches/Wireframes + Preliminary web deployment/development
5. Stakeholder review meeting - feature scope and design review R2
6. Usability test session 1
7. Finalize wireframes + Web development
8. Usability test session 2
9. Poster + Develop landing page
